## Finda-Overflow Project

- tech 커뮤니티 사이드 프로젝트
- 링크: https://findaoverflow.vercel.app/

<br />

## 사용 기술

- next.js
- firebase (firestore)
- next/auth
- react 18
- react-query
- recoil
- tailwind
